# `application/` — casos de uso

Vacía a propósito en la Fase 1.

Aquí vivirán los casos de uso (`crear solicitud`, `exportar Excel`, `duplicar
solicitud`), y ninguno está dentro del alcance aprobado. La carpeta existe
desde ahora porque el lint de límites entre capas ya la protege: cuando llegue
el primer caso de uso, la barrera ya estará puesta en vez de agregarse después,
que es cuando nunca se agrega.
